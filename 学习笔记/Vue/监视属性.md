# 监视属性

## watch

+   当被监视的属性发生变化时，回调函数自动调用，进行相关操作
+   监视的属性必须存在，才能进行监视

```js
data(){
    return {
        isHot:true
        user:{
        	age:18,
        	six:"男"
    	}
    }
}
watch:{
    isHot:{
        immediate:true,//初始化时让hadeler调用一下
        handler(newValue,oldValue){//处理函数，当isHot发生变化时
            ...
        }
    }
    //还可以监视计算属性
    //深度监视
     user:{//监视一个对象
         immediate:true,
         deep:true,//开启深度监视
         handler(newValue,oldValue){//处理函数，当isHot发生变化时
            ...
        }
     }
     // or
     'user.age':{//监视一个对象的某个属性
         handler(newValue,oldValue){//处理函数，当isHot发生变化时
            ...
        }
     }
     //当只需要handler时可以简写
     isHot(newValue,oldValue){
         .
     }
}
```
