# 绑定样式
```html
<body>
    <--绑定class样式--字符串写法，适用于：样式的类名不确定，需要动态指定-->
    <div class="basic" :class="mood" ></div>
    <--绑定class样式--数组写法，适用于：要绑定的样式不确定、名字也不确定-->
    <div class="basic" :class="classArr" ></div>
    <--绑定class样式--对象写法，适用于：要绑定的样式个数确定、名字也确定，但要动态决定用不用-->
    <div class="basic" :class="classObj" ></div>
    <--绑定style样式--对象写法-->
    <div class="basic" :style="styleObj" ></div>
</body>
<script>
    data(){
        return {
            mood:'normal',
            classArr:['class1','class2','class3'],
            classObj:{
                class1:false,
                class2:false
            },
            styleObj:{
                backgroundColor:"#ffffff",//两个词的写法
                fontSize:"40px",//两个词的写法
                color:'blue'//单个词的写法
            }
        }
    }
</script>
```
